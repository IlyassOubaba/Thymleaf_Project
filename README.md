# Thymleaf Project

---

## Description

Ce projet est une application web développée avec Spring Boot et Thymeleaf, permettant de créer une interface dynamique côté serveur.

Le but du projet est de démontrer l'utilisation de Thymeleaf comme moteur de templates pour générer des صفحات HTML dynamiques intégrées avec Spring MVC.

---

##  Technologies utilisées
- Java
- Spring Boot
- Spring MVC
-  Thymeleaf
-  H2 Database

---

## Fonctionnalités

- Affichage dynamique des صفحات HTML

- Intégration avec Spring Controller

- Gestion des données côté serveur

- Templates réutilisables avec Thymeleaf

- Architecture MVC claire
